### API Endpoints Usage

#### `/add`

f_name - first name

l_name - last name

email - user email (unique)

pwd - password



#### `/search`

query - field to search at

value - field value



#### `/update`

query - field to search at

old_value - field value

new_value - value to update



#### `/delete`

query - field to search at

value - value to delete
